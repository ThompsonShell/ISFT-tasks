# C++ Tasks — Home Assignment

A collection of 20 beginner C++ programming tasks solved as individual functions, all compiled into a single program.

## How to Build & Run

```bash
g++ main.cpp -o main
./main
```

## Structure

Each task is implemented as a separate function (`task1()` through `task20()`).  
`main()` calls them one by one in order.
